# MSc Computing Project

## Dissertation

This section contains my MSc Computing Project dissertation for the MSc in Artificial Intelligence at the University of Essex.

The dissertation focuses on the development of an AI-powered early-warning framework for the early detection and mitigation of delays and cost overruns in construction project management.

The study uses project-controls data and a transparent dashboard-based approach. It combines Earned Value Management (EVM) indicators, interpretable scoring methods, and comparative AI models to support practical and explainable decision-making.

## Code Availability

In line with the dissertation guidance, selected key aspects of the code are discussed within the dissertation, while the full implementation is provided in this repository.

The full code is organised into two main parts:

1. `dashboard_app.py`  
   Streamlit dashboard application for presenting project risk scores, alerts, EVM indicators, trends, and explainability outputs.

2. `generate_bi_from_edm.py`  
   Data-processing and BI-generation pipeline that converts raw project-control/EVM data into dashboard-ready outputs.
