## Technical Toolkit

My MSc work involved a mix of machine learning, statistics, data analysis, dashboards, knowledge modelling, and secure intelligent systems. Below is a summary of the main tools and platforms I used.

### Machine Learning and AI
- **Python**
- **scikit-learn**
- **TensorFlow / Keras**
- **LightGBM**
- **WEKA**

I used these tools to build, test, and compare machine learning models across different tasks, including classification, regression, clustering, text-based analysis, and deep learning approaches such as **CNN models** and other neural network workflows.

### Data Analysis and Statistics
- **Pandas**
- **NumPy**
- **R**
- **dplyr**

These supported data cleaning, transformation, numerical analysis, and statistical work.

### Visualisation and Reporting
- **Matplotlib**
- **Seaborn**
- **ggplot2**
- **Advanced Excel**
- **Power BI**

I used these tools for exploratory analysis, result reporting, forecasting, dashboards, and clear visual communication.

### Dashboards and Interactive Work
- **Dash / Plotly**
- **R Shiny**
- **Jupyter Notebooks**

These helped me present analysis in a more interactive and accessible way, whether through dashboards or notebook-based reporting.

### Knowledge Representation
- **Protégé**
- **OWL**
- **UML**

These were used for ontology development, structured knowledge modelling, and explainable reasoning.

### Databases, Security, and Auditability
- **SQL**
- **SQLite**
- **Paramiko**
- **SHA-256 hashing**

These supported structured storage, query-based access, secure file transfer, and integrity verification.

### Workflow and Version Control
- **Git**
- **GitHub**

These supported version control, documentation, and organised project development.

### Project Controls and Scheduling
- **Primavera P6 (Expert)**
- **Microsoft Project**

These reflect my wider background in project controls and scheduling, including planning, baseline management, progress tracking, and schedule review.
